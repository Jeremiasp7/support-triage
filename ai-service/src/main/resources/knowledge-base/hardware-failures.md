# Falhas de Hardware

## Computador não liga
**Sintomas:** pressionar o botão liga não tem efeito.
**Solução:**
1. Verificar cabo de energia e tomada.
2. Testar com outra tomada ou cabo.
3. Verificar se a fonte de alimentação está ligada (chave na traseira).
**Escalação:** Após verificações básicas sem sucesso, escalar para manutenção.

## Tela azul (BSOD) no Windows
**Sintomas:** tela azul com código de erro durante uso normal.
**Solução:**
1. Anotar o código de erro exibido.
2. Executar `sfc /scannow` para verificar arquivos do sistema.
3. Atualizar ou reverter drivers recentemente instalados.
**Escalação:** BSODs frequentes indicam falha de hardware — encaminhar para manutenção.

## HD/SSD com falha
**Sintomas:** erros de leitura/escrita, arquivos corrompidos.
**Solução:**
1. Fazer backup imediato dos dados.
2. Executar diagnóstico com CrystalDiskInfo (SMART).
3. Agendar substituição do disco se saúde estiver comprometida.
**Escalação:** Discos com status FAIL devem ser substituídos imediatamente.
